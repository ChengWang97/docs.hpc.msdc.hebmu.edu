=========
工作台
=========

:wb:`工作台`

.. raw:: html

   <hr class="divider-blue">

工作台方便用户查看和操作自己所有的实例，以及查看自己的作业运行情况，和数据管理。

:black:`账户总览`

.. raw:: html

   <hr class="divider-gray">

●	我的账户： 账户个人简要信息

●	实例概括： 创建的实例统计，包括运行中和已停止的。

●	我的作业： 在平台公共集群中提交的作业信息。

●	我创建的项目： 当前用户为管理员的项目信息以及项目中的实例统计。

●	我参与的项目： 当前用户为项目成员的项目信息以及项目中的实例统计。

●	平台应用总数： 平台已有可申请资源的应用数量。

●	我创建的实例数： 当前用户创建的实例数量，包括个人实例和项目实例。

●	平台通知： 管理员在平台上发布的通知。

●	实例资源配额：目前已经使用的实例资源和最高可申请的资源。如果要提高资源上限，需向管理员申请。

●	集群资源配额：目前已经使用的公共集群资源和最高可申请的资源。如果要提高资源上限，需向管理员申请。
